# Employee Course Editing - Quick Guide

## How to Edit Your Courses

### Step 1: Access Your Courses
1. Navigate to the **Course Builder** page
2. You'll see a list of all courses you've created

### Step 2: Find the Course to Edit
- Use the **search bar** to find courses by title, category, or instructor
- Use the **level filter** dropdown to filter by difficulty level
- Browse through your course cards

### Step 3: Start Editing
1. Click the **"Edit Course"** button on the course card
2. The course builder will open with all existing data loaded

### Step 4: Make Your Changes

#### Edit Course Information
- **Title**: Update the course name
- **Description**: Modify the course overview
- **Instructor**: Change instructor name
- **Level**: Select Beginner, Intermediate, or Advanced
- **Category**: Choose from 6 categories
- **Course Type**: Switch between Free and Paid
- **Price**: Set or update the price (₹)
- **Duration**: Update estimated completion time
- **Thumbnail**: Upload new image or paste URL
- **PDF File**: Upload course notes/materials

#### Edit Chapters
- **Add Chapter**: Click "Add Chapter" button
- **Edit Content**: Click on any chapter to expand and edit
  - Update chapter title
  - Modify content using rich text editor
  - Add or change video URL
- **Remove Chapter**: Click trash icon on chapter
- **Reorder**: Chapters are automatically numbered

#### Rich Text Editing
The content editor supports:
- **Headings**: H1, H2, H3
- **Text Formatting**: Bold, Italic, Underline
- **Lists**: Bullet points and numbered lists
- **Code Blocks**: For programming examples
- **Tables**: Structured data
- **Quotes**: Highlighted text
- **Links**: Hyperlinks to resources

### Step 5: Save Your Changes
1. Review all your edits
2. Click **"Update Course"** button (top or bottom)
3. Wait for success confirmation
4. You'll be returned to the course list

### Step 6: Verify Changes
- Check the updated course in your list
- Students will see the updated content immediately

## Tips & Best Practices

### Before Editing
- ✅ Review current course content
- ✅ Plan your changes
- ✅ Prepare new images/PDFs if needed

### While Editing
- ✅ Save frequently (click Update Course)
- ✅ Preview content formatting
- ✅ Check all chapter titles and content
- ✅ Verify video URLs work
- ✅ Ensure PDF uploads successfully

### After Editing
- ✅ Verify all changes saved correctly
- ✅ Check course appears correctly in list
- ✅ Test course from student perspective (if possible)

## Common Tasks

### Change Course from Free to Paid
1. Edit the course
2. Change "Course Type" to "paid"
3. Set the price
4. Update course

### Update Course Content
1. Edit the course
2. Expand the chapter you want to update
3. Modify the content in the rich text editor
4. Update course

### Add New Lessons
1. Edit the course
2. Click "Add Chapter"
3. Fill in title and content
4. Add video URL (optional)
5. Update course

### Replace Course Thumbnail
1. Edit the course
2. Click on thumbnail area or "Remove" button
3. Upload new image or paste URL
4. Update course

### Update Course PDF
1. Edit the course
2. Click "Remove" on existing PDF
3. Click to upload new PDF
4. Wait for upload confirmation
5. Update course

## Troubleshooting

### Changes Not Saving
- Check internet connection
- Ensure all required fields are filled
- Verify you're logged in
- Try refreshing the page

### Can't Edit a Course
- Verify you created the course
- Check if you're logged in as employee
- Contact admin if issue persists

### PDF Upload Fails
- Check file is actually PDF format
- Ensure file size is under 10 MB
- Try a different PDF file
- Check file isn't corrupted

### Thumbnail Not Showing
- Verify image URL is valid
- Check image file format (PNG, JPG, WEBP)
- Ensure file size under 5 MB
- Try uploading instead of URL

## Important Notes

⚠️ **Editing Preserves Enrollments**: Students already enrolled will remain enrolled

⚠️ **Immediate Updates**: Changes are visible to students immediately after saving

⚠️ **No Undo**: There's no undo button, so review changes before saving

⚠️ **Chapter Order**: Chapters are automatically numbered based on order

⚠️ **Required Fields**: Title, Description, Instructor, and at least one chapter are required
